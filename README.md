# BriqueParBrique

> L'essentiel du développement, une étape à la fois.

BriqueParBrique est une plateforme d'apprentissage qui décompose les nouvelles technologies en petites briques de connaissances simples, digestes et immédiatement actionnables.

Chaque parcours est conçu comme un mini-onboarding : pas de tutoriels de 10 heures, mais des chemins balisés pour comprendre les bases, installer ses outils et réaliser un premier projet concret rapidement.

**Site** : [briqueparbrique.github.io](https://briqueparbrique.github.io)

## Contribuer

Les contributions sont les bienvenues ! Consultez [CONTRIBUTING.md](CONTRIBUTING.md) pour savoir comment proposer une nouvelle brique ou améliorer le site.
